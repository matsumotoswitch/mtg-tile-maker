# MTG-TileMaker

Magic: The Gathering (MTG) 用のタイル（トークンやマーカーなど）を作成するためのWebツールです。

**[アプリを開く / Open App](https://matsumotoswitch.github.io/MTG-TileMaker/)**

## 概要 (Overview)

MTG-TileMakerは、ブラウザ上で簡単にMTG用のタイル画像を作成したり、印刷用のレイアウトを生成したりできるツールです。インストール不要ですぐに利用できます。

## 主な機能 (Features)

*   **Webベース**: ブラウザからアクセスするだけで利用可能。
*   **簡単作成**: 直感的な操作でタイルをデザイン。
*   **印刷対応**: 印刷に適した形式での出力をサポート。

## 使い方 (Usage)

1.  [MTG-TileMaker](https://matsumotoswitch.github.io/MTG-TileMaker/) にアクセスします。
2.  画面の指示に従って設定や画像選択を行います。
3.  生成された結果を保存または印刷します。

## 開発 (Development)
